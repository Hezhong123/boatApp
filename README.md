## 小船 I M 🚣 🚣 🚣
计划开发一款双语即使通讯App，在对话的场景里添加中英文互译的对话场景，
从而被动的提升我这拉垮的英文水平。

###  产品平台
* Android 
* ios
* Web系统管理端

### 技术方案
* 移动端：reactNative集成框架Expo
* 服务端：koa2+mongodb
* 系统：腾讯云contOs7.6
* 存储：阿里云Oss

### 使用工具
* sketch 原型UI
* webStorm 代码开发
* node 运行环境
* mongodbComponent 数据库管理
* android 设备
* ios 设备
